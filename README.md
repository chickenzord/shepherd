# shepherd
Rancher service watcher

### sample

```sh
docker run -d --name=shepherd coralteam/shepherd
```

### required environment

```
RANCHER_ACCESS_KEY=
RANCHER_SECRET_KEY=
RANCHER_HOST=
RANCHER_PROJECT_ID=
RANCHER_PROJECT_NAME=
SLACK_TOKEN=
SLACK_CHANNEL=
```

see `sample.env` for the complete environment variables list
